I think there's already a version of this at OLGA. My tab is quite
similar in terms of the notes used, but the strings and fret numbers 
I use are quite different. I think this way it's a lot easier to play
it so you can leave the notes to sustain a little.

This is based more on the live versions of the track than the studio
version.

There's one bit which is quite hard to pull off using just one guitar
- if you have a bass and a guitar you can split the part up to make it
work, but if you're just playing on your own it's a bit tricky so
I've transcribed an alternative part for these shapes - It just uses an
alternative shape for the same chord - it isn't exactly what's on the
record, but it works well and is nice and easy to play.
For this bit I've used curly brackets { to indicate the alternative 
lines of TAB - the top line is the 'true' line, the bottom line is
the easier one.



/  -  slide up
\  -  slide down
h  -  hammer on
p  -  pull off
